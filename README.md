# Retail Sales Dashboard

## Project Overview
Retail sales data analysis dashboard (Jan-Feb 2024)

## Data Details
- **Rows:** 50 sales records
- **Regions:** North, South, East, West
- **Categories:** Electronics, Furniture, Clothing, Groceries
- **Period:** Jan 2024 – Feb 2024

## Sheets
| Sheet | Description |
|-------|-------------|
| Sheet1 | Raw sales data |
| Monthly Sales Trend | Month-wise pivot |
| Profit by Category | Category-wise profit |
| Sales by Region | Region-wise sales |
| Dashboard | Visual dashboard |

## Key Metrics
- Total Sales: ₹1,27,500
- Total Profit: ₹25,500
- Profit Margin: 20%
